## Введение
`DostavistoVrotBeristo` — это упрощенный прототип **приложения для заказа еды из ресторанов**. Основная цель продукта, предоставить пользователю интуитивно понятный интерфейс для просмотра меню, добавления блюд в корзину и оформления заказа.

## Примеры использования
`10 ключевых функций приложения`:
1.  При вводе номера телефона и нажатии кнопки "Получить код" система отправляет SMS с подтверждением.
2.  При вводе названия в строке поиска ("Бургерная") отображаются только подходящие здания.
3.  При нажатии на кнопку "Вегетарианское" из списка блюд скрываются все позиции, содержащие мясо.
4.  При нажатии на кнопку "Добавить" рядом с продуктом счетчик товаров в иконке корзины увеличивается на 1.
5.  При нажатии на кнопку "Убрать" рядом с продуктом счетчик товаров в иконке корзины уменьшается на 1.
6.  **При вводе текста "`IgorLox`" в поле "Промокод" и нажатии "Применить" итоговая стоимость заказа делается со скидкой.**
7.  При нажатии на кнопку "Заказать" на экране появляется сообщение "Ваш заказ передан в ресторан".
8.  При переключении кнопки с "Наличными" на "Картой" появляется поле для ввода номера карты.
9.  При нажатии на заказы в разделе "Истории" открывается вся история ваших заказов.
10. При нажатии на кнопку "Заказать заново" в истории заказов все позиции из прошлого заказа автоматически добавляются в текущую корзину.

## Технические требования
Программное и аппаратное обеспечение:
*   Операционная система: `Хотяб сенсорный телефон`
*   Браузер: Не обязателен (google и т.д.)
*   Интернет-соединение: Минимальная скорость для загрузки блюд.
